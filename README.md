# mini-redeSocial
Projeto feito para o Checkpoint 3 da matéria de Web Development de Engenharia de Software na FIAP.

## Funcionalidades

- Criação de posts com texto, imagens e categoria
- Edição de posts
- Exclusão de posts
- Filtro de posts por categoria
